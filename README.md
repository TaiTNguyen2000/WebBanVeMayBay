# ASP.NET MVC 5 Flight Ticket Booking Website with SQL Server

Features:

- Customer:
  - View news articles
  - Sign up / Sign in
  - Edit profile
  - Search for flights
  - Book flights
- Admin:
  - Manage flights
  - Manage airports
  - Manage airline companies
  - Manage bookings
  - Manage users
  - Manage news
- Staff:
  - Manage bookings
  - View users
- Airline company:
  - Manage their own flights

Installation guide:

1.  Pull code
2.  Create QuanLyBanVeMayBay database in SQL Server
3.  Restore .bak file
4.  Open solution in Visual Studio
5.  Change connection string
